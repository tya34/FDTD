# FDTD project

This repository is a curated handoff for the FDTD work under `C:\еҚҡеЈ«\иҜҫйўҳ1\FDTD`.

## Current scope

The uploaded repository keeps reproducible and reviewable project files first:

- FDTD scripts exported as text files for Arch, Helix, Ring, Taper, Tube, and deformation variants.
- Blender/Python source scripts used to build or render model geometry.
- Work logs and next-step notes for continuing the project on another computer.

Large binary simulation/model artifacts are intentionally not uploaded in this initial pass. The local project contains about 1.0 GB of `.fsp`, `.blend`, and `.png` files. These are better handled with Git LFS, cloud storage, or a separate release/archive workflow.
